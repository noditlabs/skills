# polygon-eth_gasprice

> eth_gasPrice

- **Category**: Node API - Polygon (JSON-RPC)

## Supported Chains

Polygon (mainnet, amoy)

## Method

`eth_gasPrice`

## Parameters

None - this method takes no parameters.

## Returns

- **result** (`string`): An integer encoded as a hexadecimal string (e.g., `'0x1a'`).

## Example Request

```json
{
  "id": 1,
  "jsonrpc": "2.0",
  "method": "eth_gasPrice"
}
```
