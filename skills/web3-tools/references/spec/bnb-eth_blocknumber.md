# bnb-eth_blocknumber

> eth_blockNumber

- **Category**: Node API - Bnb (JSON-RPC)
- **Official Docs**: https://developer.nodit.io/reference/bnb-eth_blocknumber

## Supported Chains

Bnb (mainnet, testnet)

## Method

`eth_blockNumber`

## Parameters

None - this method takes no parameters.

## Returns

- **result** (`string`): An integer encoded as a hexadecimal string (e.g., `'0x1a'`).

## Example Request

```json
{
  "id": 1,
  "jsonrpc": "2.0",
  "method": "eth_blockNumber"
}
```
