# solana-getRecentPerformanceSamples

> getRecentPerformanceSamples

- **Category**: Node API - Solana (JSON-RPC)

## Supported Chains

Solana (mainnet, devnet)

## Method

`getRecentPerformanceSamples`

## Parameters

Array of 0-1 item(s). Number of samples to return (maximum 720)

## Returns

An array of objects. 

## Example Request

```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "getRecentPerformanceSamples",
  "params": [
    2
  ]
}
```
