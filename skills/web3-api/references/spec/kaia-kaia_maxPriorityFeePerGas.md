# kaia-kaia_maxPriorityFeePerGas

> Returns the current max priority fee per gas in wei (EIP-1559).

- **Category**: Node API - Kaia (JSON-RPC)
- **Official Docs**: https://developer.nodit.io/reference/kaia-kaia_maxPriorityFeePerGas

## Supported Chains
kaia (mainnet, kairos)

## Method
`kaia_maxPriorityFeePerGas`

## Parameters
None - this method takes no parameters.

## Returns
- **maxPriorityFeePerGas** (`string`): Max priority fee per gas as a hex string in wei.
