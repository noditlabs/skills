# kaia-klay_newBlockFilter

> Creates a filter to notify when new blocks are created. Use kaia_getFilterChanges to poll.

- **Category**: Node API - Kaia (JSON-RPC)
- **Official Docs**: https://developer.nodit.io/reference/kaia-klay_newBlockFilter

## Supported Chains
kaia (mainnet, kairos)

## Method
`klay_newBlockFilter`

## Parameters
None - this method takes no parameters.

## Returns
- **filterId** (`string`): Hex filter ID for use with kaia_getFilterChanges.
