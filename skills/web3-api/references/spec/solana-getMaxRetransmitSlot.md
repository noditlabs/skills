# solana-getMaxRetransmitSlot

> Get the max slot seen from retransmit stage

- **Category**: Node API - Solana (JSON-RPC)
- **Official Docs**: https://developer.nodit.io/reference/solana-getMaxRetransmitSlot

## Supported Chains
solana (mainnet, devnet)

## Method
`getMaxRetransmitSlot`

## Parameters
None.

## Returns
An integer (uint64): The max slot seen from retransmit stage.

## Example Request
```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "getMaxRetransmitSlot"
}
```
