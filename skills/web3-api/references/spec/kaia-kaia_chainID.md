# kaia-kaia_chainID

> Returns the chain ID of the connected network as a hex string (EIP-155).

- **Category**: Node API - Kaia (JSON-RPC)
- **Official Docs**: https://developer.nodit.io/reference/kaia-kaia_chainID

## Supported Chains
kaia (mainnet, kairos)

## Method
`kaia_chainID`

## Parameters
None - this method takes no parameters.

## Returns
- **chainId** (`string`): The chain ID as a hex string.
