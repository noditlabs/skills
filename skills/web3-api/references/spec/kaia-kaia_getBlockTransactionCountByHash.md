# kaia-kaia_getBlockTransactionCountByHash

> Returns the number of transactions in a block identified by block hash.

- **Category**: Node API - Kaia (JSON-RPC)
- **Official Docs**: https://developer.nodit.io/reference/kaia-kaia_getBlockTransactionCountByHash

## Supported Chains
kaia (mainnet, kairos)

## Method
`kaia_getBlockTransactionCountByHash`

## Parameters
- **blockHash** (`string`): 64-char hex block hash.

## Returns
- **count** (`string`): Number of transactions as a hex string.
