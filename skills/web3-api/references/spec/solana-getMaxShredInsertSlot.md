# solana-getMaxShredInsertSlot

> Get the max slot seen from after shred insert

- **Category**: Node API - Solana (JSON-RPC)
- **Official Docs**: https://developer.nodit.io/reference/solana-getMaxShredInsertSlot

## Supported Chains
solana (mainnet, devnet)

## Method
`getMaxShredInsertSlot`

## Parameters
None.

## Returns
An integer (uint64): The max slot seen from after shred insert.

## Example Request
```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "getMaxShredInsertSlot"
}
```
