# kaia-kaia_newPendingTransactionFilter

> Creates a filter to notify when new pending transactions arrive. Use kaia_getFilterChanges to poll.

- **Category**: Node API - Kaia (JSON-RPC)
- **Official Docs**: https://developer.nodit.io/reference/kaia-kaia_newPendingTransactionFilter

## Supported Chains
kaia (mainnet, kairos)

## Method
`kaia_newPendingTransactionFilter`

## Parameters
None - this method takes no parameters.

## Returns
- **filterId** (`string`): Hex filter ID for use with kaia_getFilterChanges.
