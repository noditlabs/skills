# kaia-kaia_gasPrice

> Returns the current estimated gas price in wei.

- **Category**: Node API - Kaia (JSON-RPC)
- **Official Docs**: https://developer.nodit.io/reference/kaia-kaia_gasPrice

## Supported Chains
kaia (mainnet, kairos)

## Method
`kaia_gasPrice`

## Parameters
None - this method takes no parameters.

## Returns
- **gasPrice** (`string`): Current gas price as a hex string in wei.
