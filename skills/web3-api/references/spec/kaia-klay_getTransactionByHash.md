# kaia-klay_getTransactionByHash

> Returns transaction information for a given transaction hash.

- **Category**: Node API - Kaia (JSON-RPC)
- **Official Docs**: https://developer.nodit.io/reference/kaia-klay_getTransactionByHash

## Supported Chains
kaia (mainnet, kairos)

## Method
`klay_getTransactionByHash`

## Parameters
- **transactionHash** (`string`): 64-char hex transaction hash.

## Returns
- **transaction** (`object|null`): Transaction object or `null` if not found.
