# solana-getGenesisHash

> Returns the genesis hash

- **Category**: Node API - Solana (JSON-RPC)
- **Official Docs**: https://developer.nodit.io/reference/solana-getGenesisHash

## Supported Chains
solana (mainnet, devnet)

## Method
`getGenesisHash`

## Parameters
None.

## Returns
A string: the genesis hash as a base-58 encoded string.

## Example Request
```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "getGenesisHash"
}
```
